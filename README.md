# newsLookup

## [1] Installation

1. git clone https://github.com/Thomas-Leung-852/newsLookup.git
2. npm install

## [2] Seach by keywords

node index.js "Hong Kong,Chow"

## [3] Get the news websites list from a source 

node index.js -geturl "https://www.w3newspapers.com/newssites/"
